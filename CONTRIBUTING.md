# 贡献指南

感谢你考虑为 agent-compass 做出贡献。

## 贡献方式

### 报告问题

如果你发现文档错误、逻辑不一致或改进建议，请提交 Issue。

### 提交改进

1. Fork 本仓库
2. 创建你的特性分支（`git checkout -b feature/amazing-idea`）
3. 提交你的修改（`git commit -m 'Add: 描述你的改动'`）
4. 推送到分支（`git push origin feature/amazing-idea`）
5. 打开 Pull Request

### 文档规范

本项目遵循自身的命名规范。所有文档使用：

```
L{Level}_{Category}{NN}_{Slug}_{Title}.md
```

详见 `templates/L1_A02_naming-convention_命名规范.md`。

## 贡献范围

欢迎以下类型的贡献：

- **方法论补充**：你实践过的、对 Agent 友好的开发实践
- **模板改进**：让模板更通用、更易用
- **案例分享**：你用 agent-compass 做过的项目经验
- **翻译**：将文档翻译为其他语言
- **Bug 修复**：文档中的错误、断链、不一致
