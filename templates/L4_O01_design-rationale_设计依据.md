## AGENT: 初始化流程

```
1. 执行 git log --oneline --all -50
   如果 git 不可用 → 检查 CHANGELOG.md / PR 描述 / 记忆中的对话
2. 筛选含以下关键词的 commit: "选择"/"改用"/"不用"/"因为"/"why"/"decision"
3. 反推每条为一行设计依据: 决策 | 来源 | 证据
4. 如果 git log + CHANGELOG 都无信息 → 标注 [待人类补充]
5. 日常维护: 每次做新决策 → 追加一行
```

## 完成标准

□ 表格 ≥ 3 行
□ 每行有来源（测试/评审/用户反馈）
□ 有 ≥ 1 行记录了替代方案
□ 最近 1 个月的技术决策都在表中

## 反模式

| 不要 | 要 |
|------|------|
