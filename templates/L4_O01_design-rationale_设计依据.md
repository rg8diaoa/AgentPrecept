## AGENT: 初始化流程

```
1. 执行 git log --oneline --all -50
2. 筛选含以下关键词的 commit: "选择"/"改用"/"不用"/"因为"/"why"/"decision"
3. 反推每条为一行设计依据: 决策 | 来源 | 证据
4. 如果 git log 信息不足 → 标注 [待人类补充]
5. 日常维护: 每次做新决策 → 追加一行
```
